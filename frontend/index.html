RAG System v1.0.0

Fully offline-capable Retrieval-Augmented Generation chatbot.

Getting Started
  1. Ensure backend is running on port 8000.
  2. Configure your preferred LLM in Settings.
  3. Upload PDF documents via the left sidebar.
  4. Ask questions! The system will ground answers in the PDFs.
Uploading Documents
  • Supported formats: PDF only.
  • Maximum 4 files per upload batch to prevent overwhelming the local embedding model.
  • Documents are chunked and embedded automatically. Deduplication is handled by MD5 hash.
  • To replace a document, delete it from the list and upload it again.
